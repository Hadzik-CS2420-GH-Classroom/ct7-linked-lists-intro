# ct7-linked-lists-intro

Code-Together 7: Building Blocks - Nodes & Basic Singly Linked List

Covers:

- How to form a proper node (task 1)
- Singly linked list benefits/drawbacks vs arrays (task 2.1)
- Inserting in the front and back (task 2.2)
- Removing from the front (task 2.3)

- Activity outline:
- Start with a discussion comparing arrays vs linked lists (memory layout, resize cost, random access tradeoffs)
- Define a templated Node struct with data and a next pointer
- Begin a SinglyLinkedList class with head pointer and basic constructor/destructor
- Implement push_front(), push_back(), and pop_front()
- Write a simple print() method to verify everything works
